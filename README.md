# Task Manager - Консольное приложение для управления задачами

**Автор:** Иван Иванов

## Описание программы

Task Manager - это консольное приложение для управления задачами, разработанное на Python с использованием принципов объектно-ориентированного программирования и паттерна MVC (Model-View-Controller).

### Основные возможности:

- ✅ **Создание, редактирование и удаление задач**
- 📊 **Приоритеты задач** (Low, Medium, High)
- 🔄 **Изменение статуса задачи** (To Do, In Progress, Done)
- 🎯 **Фильтрация задач** по статусу и приоритету
- ⏪ **Отмена последнего действия** (стек отмены)
- 🎯 **Очередь приоритетов** - автоматическая сортировка по важности
- 💾 **Сохранение и загрузка** задач в/из JSON файла
- 🚨 **Срочные задачи** с дедлайном (расширенный тип задач)
- 🛡️ **Валидация ввода** - защита от некорректных данных

### Технические особенности:

- Инкапсуляция (закрытые поля, геттеры/сеттеры)
- Наследование (класс UrgentTask наследуется от Task)
- Полиморфизм (переопределение методов)
- Сериализация/десериализация в JSON
- Паттерн MVC для разделения логики

## Установка и запуск

### Требования:
- Python 3.6 или выше
- Стандартные библиотеки Python (не требуют дополнительной установки)

1. **Сохраните код** в файл с именем `task_manager.py`

2. **Откройте терминал/командную строку** в папке с файлом

3. **Запустите приложение:**
   ```bash
   python task_manager.py
